/posts/{id}

Getting Started

To interact with this API, you can use tools like curl, Postman, or any HTTP client. All endpoints return JSON responses (except this home page).

Example Usage:

curl -X GET http://localhost:8080/posts
curl -X POST http://localhost:8080/posts/like?id=1
curl -X POST -H "Content-Type: application/json" -d '{"title":"Hello World","content":"My first post"}' http://localhost:8080/posts
